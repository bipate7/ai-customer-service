## 🚀 Quick Deploy

1. **Push to GitHub**
2. **Go to [vercel.com](https://vercel.com)**
3. **Import GitHub repository**
4. **Click Deploy** (takes 2 minutes)

Your chatbot will be live at: `your-project.vercel.app`
